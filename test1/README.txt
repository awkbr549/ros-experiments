#####
How to Run
#####
To compile the code, in the zip home directory, use the following commands:
   $ ros_core
   $ catkin_make clean
   $ catkin_make

To run the code with rosrun, use the following
   $ rosrun my_turtle Move_Turtle
or
   $ rosrun stopper AvoidObst_Turtle

To run the code with roslaunch, use the following
   $ roslaunch my_turtle move_turtle.launch
or
   $ roslaunch stopper obst_turtle.launch
